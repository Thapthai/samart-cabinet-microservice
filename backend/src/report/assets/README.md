# ฟอนต์ไทยสำหรับรายงาน PDF

## ไฟล์ที่ต้องมีในโฟลเดอร์นี้

- **`THSarabunNew.ttf`** – ฟอนต์ปกติ (ถ้าไม่มี PDF จะแสดงภาษาไทยไม่ได้)
- **`THSarabunNew Bold.ttf`** – ฟอนต์หนา (ถ้าไม่มีจะใช้ฟอนต์ปกติแทน)

ดาวน์โหลดได้จากชุดฟอนต์ TH Sarabun New ของรัฐบาลไทย หรือจากแหล่งที่ให้ใช้ฟอนต์ไทยได้ตามลิขสิทธิ์

เมื่อวางไฟล์แล้ว ให้ restart backend (หรือรัน `npm run build` ใน backend เพื่อ copy โฟลเดอร์นี้ไปที่ dist)
